Firebase

  • Use firebase to connect to other Google services
  • Firebase functions works as classic functions
  • Firebase functions can be triggered by events

Packages

  • firebase : client package
  • firebase-admin : server package

Services

  • Auth
  • Analytics, Crashlytics
  • Pub/Sub
  • Cloud storage
  • Databases (Firebase, BigQuery, ...)
  • Firebase Cloud Messaging
  • Webhook
