A function

  • JavaScript
  • Express.JS handler (or module)
  • Compatible with public and private NPM packages
    • exports.hello = (req, res) => {
    res.send(`Hello World!`);
}

Deploy a function

    gcloud functions deploy func-name --trigger-http
curl https://region-projectId.cloudfunctions.net/func-name

Function settings

  • Possibility to set the region where the function will be deployed
  • Possibility to set the memory allocated to the function (default : 256mb)
  • Possibility to set the timeout of the function (default : 60s)
