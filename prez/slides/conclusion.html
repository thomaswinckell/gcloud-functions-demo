Pros

  • No setup, straightforward
  • Automatic scaling
  • Monitoring, reporting
  • Easy to connect with a lot of services
  • Chain services easily
  • Nice and easy to use UI

Cons

  • Still in Beta
  • Documentation not always up-to-date
  • Environment locked, cannot install native modules
  • Difficult to be outside of the use cases suggested by Google Cloud
  • Architecture should be designed specifically for this cloud platform

Conclusion

  • Can speed up product development
  • Not ultra flexible
  • My advice : do a POC with all corner cases before choosing it
