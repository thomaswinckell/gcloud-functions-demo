Demo

  • User upload a file (mp3) to an HTTP function
  • Function upload the file to cloud storage
  • When a file is created on cloud storage, get track metadata and push them to firebase
  • When an artist/album is created in firebase, get artwork and upload them in cloud storage
  • When a track is created in firebase, update algolia index (for search)
  • Send push notifications when there is a new track
  • User can play the song: call an HTTP function that stream from cloud storage
